Map API

One of the coolest parts of this application is the map API that we implemented. Initially we wanted to use Google Maps and have it be able to track the properties based on cities automatically.


Unforunatly, Google likes to charged for such features so we were forced to look else where. We found Mapbox API was free and has been used in many ember applications, so after watching a few tutorials we were able to implement it.


The only ussue with the free version is the user needs to insert all the lat and lng points maunally to find the locations on the map.

Filter

Another interesting feature with this application is the use of a search bar which allows the user to type in a city and the program will automatically perform a search query and look through all the objects in the JSON database to see which city matches the query.

Conclusion

Ember has solidified itself as one of the greatly optimized frameworks throughout the years. Although it falls short in comparison to the popularities of frameworks such as React or Angular, Ember's clean approach to many topics allows users to freely explore their creative mind. Creating apps and websites with a much cleaner-looking code, and functionalities to help ease the process of coding some tedious tasks.


With this project, we hope users are able to understand the capabilities of Ember and utilize it to its full potential!
